# Smash Drop 🍔🎮

Um jogo web mobile desenvolvido para uma hamburgueria com o objetivo de entreter os clientes enquanto aguardam seus pedidos.

O projeto foi desenvolvido como freela utilizando **Phaser.js**, com foco em:

* experiência rápida e divertida;
* compatibilidade mobile;
* identidade visual personalizada da hamburgueria;
* gameplay simples e intuitivo.

---

## 📱 Sobre o Projeto

No jogo, o jogador controla o personagem **Jimmy**, que deve pegar os ingredientes e comidas corretas dos pedidos dos clientes enquanto desvia de objetos perigosos.

O sistema inclui:

* fases progressivas;
* geração dinâmica de pedidos;
* pontuação;
* sistema de vidas;
* controles touch para dispositivos móveis;
* interface personalizada da marca;
* telas de menu e game over.

